🔌 Ενσωμάτωση Chatbot στο eifron-diabetes.gr (WordPress)

Το chatbot τρέχει στο: https://eifronchatbot.onrender.com

Στόχος: να εμφανίζεται ένα floating 💬 κουμπί σε κάθε σελίδα του WordPress site.

Μέθοδος 1: Plugin «WPCode» (Συνιστάται — Πιο εύκολο)

Χωρίς κώδικα, μέσα από το WordPress admin

  1. WordPress Dashboard → Plugins → Add New
  2. Αναζητήστε «WPCode» (ή «Insert Headers and Footers»)
  3. Εγκατάσταση → Ενεργοποίηση
  4. Πηγαίνετε: Code Snippets → Header & Footer
  5. Στο πεδίο «Footer» επικολλήστε:
<script src="https://eifronchatbot.onrender.com/static/widget.js"></script>
  1. Πατήστε Save Changes
  2. Ανοίξτε το site σε incognito tab — θα δείτε το 💬 κάτω δεξιά

Μέθοδος 2: functions.php (Για developers)

Μέσω κώδικα στο theme

WordPress Dashboard → Appearance → Theme File Editor → functions.php

Προσθέστε στο τέλος:

// Eifron Chatbot Widget
function eifron_chatbot_widget() {
    echo '<script src="https://eifronchatbot.onrender.com/static/widget.js"></script>';
}
add_action('wp_footer', 'eifron_chatbot_widget');
⚠️ Αν χρησιμοποιείτε child theme, βάλτε τον κώδικα στο functions.php του child theme. Αλλιώς, μια ενημέρωση theme θα σβήσει τις αλλαγές σας.

Μέθοδος 3: Elementor (Αν χρησιμοποιείτε Elementor)

Μέσω Elementor HTML widget

  1. Elementor → Templates → Theme Builder → Footer (ή δημιουργήστε νέο)
  2. Προσθέστε HTML widget
  3. Επικολλήστε:
<script src="https://eifronchatbot.onrender.com/static/widget.js"></script>
  1. Publish → Condition: Entire Site

Προσαρμογή (Προαιρετικό)

Αν θέλετε να αλλάξετε χρώμα ή θέση, βάλτε πριν το script:

<script>
  window.EIFRON_CHAT_CONFIG = {
    position: "right",       // "right" ή "left"
    color: "#0d6e3f",        // χρώμα κουμπιού (πράσινο Eifron)
  };
</script>
<script src="https://eifronchatbot.onrender.com/static/widget.js"></script>

Μόνο σε συγκεκριμένες σελίδες (Προαιρετικό)

Αν θέλετε το chatbot μόνο σε ορισμένες σελίδες, χρησιμοποιήστε conditional loading στο functions.php:

function eifron_chatbot_widget() {
    // Μόνο στην αρχική και στη σελίδα επικοινωνίας
    if ( is_front_page() || is_page('contact-us') ) {
        echo '<script src="https://eifronchatbot.onrender.com/static/widget.js"></script>';
    }
}
add_action('wp_footer', 'eifron_chatbot_widget');

Ή σε ΟΛΕΣ τις σελίδες εκτός από μία:

function eifron_chatbot_widget() {
    if ( ! is_page('checkout') ) {  // Όχι στο checkout
        echo '<script src="https://eifronchatbot.onrender.com/static/widget.js"></script>';
    }
}
add_action('wp_footer', 'eifron_chatbot_widget');

✅ Έλεγχος

3. Πρέπει να δείτε 💬 κάτω δεξιά
4. Κλικ → ανοίγει chat → δοκιμάστε «σφάλμα patch» ή browse κατηγορίες

Troubleshooting

Δεν εμφανίζεται το widget:

  • Ελέγξτε browser console (F12 → Console) για errors
  • Βεβαιωθείτε ότι δεν υπάρχει Content Security Policy που μπλοκάρει external scripts
  • Δοκιμάστε σε incognito (cache/extensions μπορεί να εμποδίζουν)
  • Αν χρησιμοποιείτε caching plugin (WP Rocket, LiteSpeed κλπ), κάντε purge cache

Αργή πρώτη φόρτωση (30-60 sec):

  • Φυσιολογικό στο Render Free tier — ο server «ξυπνάει»
  • Λύση: StatusCake monitor (βλ. κύριο οδηγό) ή αναβάθμιση σε Render Starter ($7/μήνα)
