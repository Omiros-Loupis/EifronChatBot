# Eifron Diabetes FAQ Chatbot

Chatbot υποστήριξης για το https://eifron-diabetes.gr/  
Αντλία S9 / EasyPatch / CGM αισθητήρα — 68 ερωτοαπαντήσεις.

## Live
https://eifronchatbot.onrender.com

## Local Setup
```bash
pip install -r requirements.txt
python chatbot.py
# → http://localhost:5070
```

## WordPress Ενσωμάτωση
Βλ. `embed_example.html` — 3 μέθοδοι (WPCode plugin, functions.php, Elementor).

Η πιο γρήγορη: WPCode plugin → Footer → επικολλήστε:
```html
<script src="https://eifronchatbot.onrender.com/static/widget.js"></script>
```
